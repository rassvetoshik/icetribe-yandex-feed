# YML-фид IceTribe для Яндекс Директа

Репозиторий автоматически строит отдельный YML-фид из [исходного фида Tilda](https://icetribe.ru/tstore/yml/6aaa63d2ffe6f090367e6716269e1ab6.yml) и публикует его через GitHub Pages.

Публичный адрес после первого успешного запуска:

`https://rassvetoshik.github.io/icetribe-yandex-feed/yandex-direct.yml`

## Что попадает в фид

- 5 вариантов инфракрасной сауны с ПЭМП — с цветом и размером в названии;
- 3 ванны для закаливания;
- инфракрасная кепка для роста волос;
- Энерджи ПЭМП-мат, Офис ПЭМП-мат и ПЭМП-пояс;
- Супер Редлайт и Мега Редлайт;
- LED-щётка с двумя сменными насадками.

Всего — 15 предложений. Для каждого сохраняются исходные `offer id`, цена, URL, изображения и другие параметры. Меняются только состав фида и названия товаров.

## Обновление

GitHub Actions пересобирает фид каждый час в 17 минут после часа и при каждом изменении основной ветки. При невозможности получить исходный фид или при отсутствии одного из 15 согласованных ID сборка завершается ошибкой и не публикует неполный файл.

## Локальная проверка

```bash
npm test
npm run build
```

Результат локальной сборки: `docs/yandex-direct.yml`.
