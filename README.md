# YML-фид IceTribe для Яндекс Директа

Репозиторий автоматически строит отдельный YML-фид из [исходного фида Tilda](https://icetribe.ru/tstore/yml/6aaa63d2ffe6f090367e6716269e1ab6.yml) и публикует его через GitHub Pages.

Публичный адрес после первого успешного запуска:

`https://rassvetoshik.github.io/icetribe-yandex-feed/yandex-direct.yml`

Отдельный фид для блока «Страницы каталога» в Директе:

`https://rassvetoshik.github.io/icetribe-yandex-feed/catalog-pages.yml`

## Что попадает в фид

- 5 вариантов инфракрасной сауны с ПЭМП — с цветом и размером в названии;
- 3 ванны для закаливания;
- инфракрасная кепка для роста волос;
- Энерджи ПЭМП-мат, Офис ПЭМП-мат и ПЭМП-пояс;
- Супер Редлайт и Мега Редлайт;
- LED-щётка с двумя сменными насадками.

Сейчас в фиде 13 актуальных предложений. Для каждого сохраняются исходные `offer id`, цена, URL, изображения и другие параметры. Меняются только состав фида и названия товаров.

21 июля 2026 года Tilda перестала передавать чёрную сауну M и панель «Мега Редлайт». Эти предложения исключены из обновляемых файлов, без автоматической замены на другие товары. Ссылки на оба фида при этом остаются прежними.

В `catalog-pages.yml` те же 13 предложений дополнительно привязаны к шести реальным разделам страницы `https://icetribe.ru/katalog`: ванны, ПЭМП-маты, инфракрасные сауны, кепка, LED-щётка и панели красного света. Этот файл содержит обязательные для Директа элементы `<collections>` и `<collectionId>`.

## Обновление

GitHub Actions пересобирает фид каждый час в 17 минут после часа и при каждом изменении основной ветки. При невозможности получить исходный фид или при отсутствии одного из 15 согласованных ID сборка завершается ошибкой и не публикует неполный файл.

## Локальная проверка

```bash
npm test
npm run build
```

Результаты локальной сборки: `docs/yandex-direct.yml` и `docs/catalog-pages.yml`.
